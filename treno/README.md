# Agenda Treno 4ITA — versione definitiva

Pagina statica GitHub Pages per gestire l'assegnazione del Treno dell'alleanza 4ITA in Last War.

## Struttura

Carica la cartella `treno` nel repository `tattiche`:

```text
tattiche/
  index.html
  treno/
    index.html
    assets/
      4ita_logo.png
```

URL previsto:

```text
https://lucaangel.github.io/tattiche/treno/
```

## Note

- Nessun backend.
- Nessuna libreria esterna.
- HTML, CSS e JavaScript vanilla.
- Salvataggio locale tramite localStorage.
- Password eliminazione righe storico/log: `0000`.
